# TechNewsAgg
An android app that displays tech news from multiple sources.

To use the application, the user must create an account that will be stored in a local database.
Once he created the account, he can view the most recent articles from the available publications.
An user will only be shown articles from the sources that he subscribed to, these subscriptions
are stored using Shared Preferences and are bound to his account.

The articles that can be found using this application belong to romanian or international tech publications/websites.
Once an article is selected, the app will open the article using its original link in the default browser,
the user will be redirected to the corresponding website that the article belongs to.

### Current supported sources :
* The Verge
* Tom's Hardware
* Venture Beat
* SlashGear
* The Indian Express
* Tech Radar
* Zona IT ( romanian )
* Yoda ( romanian)
* PlayTech ( romanian )
